## Module Summaries

### Module 1 – Presentation Software & Spreadsheets
Introduce us to how a presentation is made using a digital tool like Canva, Prezi or Miro.Also includes fundamentals of electronic spreadsheets — functions, formulas, charts, and graphics in Microsoft Excel.

### Module 2 – Programming & Collaborative Learning Platforms
Introduction to collaborative tools like Google Docs, Slides, Sheets, and Forms. Also includes, Coding practice platforms HackerRank, CodeChef, HackerEarth and CodeTantra, along with virtual laboratories.

### Module 3 – Building a Digital Portfolio
Teaches us how to create a professional online presence using platforms such as LinkedIn, GitHub, Kaggle, ResearchGate, Stack Overflow, TechGig, and others. This module is useful for career. 

### Module 4 – Email Etiquette & Social Media Etiquette
Focuses on professional email communication — subject lines, greetings, tone, font choice, signatures, attaching files, and replying correctly. Also introduces the basics of responsible social media use.

### Module 5 – Cybercrime & Financial Fraud Awareness
Describes types of cybercrimes including phishing, identity theft, UPI fraud, and cyberbullying. Includes prevention measures, digital hygiene practices, social media etiquette, and online certification platforms.
